**House of Interiors — AI Agent Dashboard**

A full-featured business management PWA for House of Interiors, Noida.

## Features
- ✦ **AI Agent** — Smart assistant for all daily tasks
- ◈ **CRM & Lead Tracker** — Pipeline, follow-ups, status tracking
- ≋ **BOQ Builder** — Instant AI-powered project estimates
- ⬡ **Social Media** — Caption generator, content calendar
- ◻ **Office Tasks** — Daily task manager with AI briefing

## How to deploy
### GitHub Pages (Free hosting)
1. Upload all files to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your app will be live at `https://yourusername.github.io/repo-name`

### Add to Phone (Android)
1. Open the GitHub Pages URL in Chrome
2. Tap ⋮ → **"Add to Home Screen"** or **"Install App"**

### Add to Phone (iPhone)
1. Open the URL in Safari
2. Tap Share → **"Add to Home Screen"**
