## 🚀 Deploy on GitHub Pages (Free)

### Step 1 — Create repository
1. Go to **github.com** → Sign in
2. Click **"New"** (green button, top left)
3. Name it: `hoi-website`
4. Set to **Public**
5. Click **"Create repository"**

### Step 2 — Upload files
1. Click **"uploading an existing file"**
2. Drag all 6 files into the upload area
3. Click **"Commit changes"**

### Step 3 — Enable GitHub Pages
1. Go to **Settings** tab in your repo
2. Click **"Pages"** in the left sidebar
3. Under Source → select **"main"** → **"/ (root)"**
4. Click **Save**
5. Wait 1–2 minutes → your site is live! ✅

### Your live URL will be:
`https://yourusername.github.io/hoi-website`

## 📱 Add to Phone as App
1. Open the live URL in Chrome (Android) or Safari (iPhone)
2. **Android:** Tap ⋮ → "Add to Home Screen"
3. **iPhone:** Tap Share → "Add to Home Screen"

## 🔗 Custom Domain (optional)
To use `www.houseof-interiors.com` instead of the GitHub URL:
1. In GitHub Pages settings → add your custom domain
2. In your domain registrar (GoDaddy etc.) → add a CNAME record pointing to `yourusername.github.io`
